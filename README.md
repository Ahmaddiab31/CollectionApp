## Установка <br>
Для установки программы нужно зайти в раздел releases и установить .apk из самого нового релиза,
после чего найти скачанный файл в своём файловом менеджере и нажать на него. В открывшемся окне 
подтвердить установку После завершения процесса окно закроется и приложение будет установлено на устройство.<br>
## Использование<br>
Приложение предназначено для работа с коллекции приложение позволяет создать коллекции и добавит элементы в коллекции и также экспорт и удаление коллекции<br>
## Разработка<br>
Данный проект не нуждается в посторонней помощи для разработки, так как он создаётся в рамках учебной практики, что подразумевает его разработку одним человеком.<br>
